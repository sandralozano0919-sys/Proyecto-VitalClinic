# VitalClinic

**VitalClinic** es una aplicación web para la gestión de citas médicas, desarrollada en **Visual Studio**. La plataforma permite registrar pacientes, asignar citas con especialistas y llevar un control de los horarios disponibles, optimizando la administración de una clínica o consultorio.

## Contenido del repositorio

- Código fuente completo del proyecto en Visual Studio.
- Archivos de configuración del backend y frontend.
- Scripts necesarios para la instalación y ejecución.
- Carpeta con estilos y recursos estáticos (CSS, imágenes).

## Tecnologías utilizadas

- **Backend:** Django / Python
- **Frontend:** HTML, CSS, JavaScript
- **Control de versiones:** Git / GitHub
